## 🥇 Featured Active Projects

The following projects are actively updated, have high GitHub community recognition (Stars), cover major platforms, and offer friendly learning curves for beginners:

### 1. [FlClashX](https://github.com/pluralplay/FlClashX/releases)
* **Target Audience:** Cross-platform users who want a modern UI, uniform coverage, and a "no-fuss" experience.
* **Core Advantages:** Fluent Design, beautiful interface; strong multi-platform synchronization; auto-aggregation of subscriptions and node speed testing.

### 2. [Clash Verge Rev](https://github.com/clash-verge-rev/clash-verge-rev/releases)
* **Target Audience:** Users migrating from CFW who value advanced desktop routing, Tun mode, and ecosystem continuity.
* **Core Advantages:** One of the most active desktop branches currently; perfectly inherits the Clash Verge ecosystem; provides stable system-level integration (Win/Mac/Linux).
* **Tutorial:** [Clash Verge Rev Installation & Configuration Guide](https://openetkit.com/en/clash-verge-rev/install)

### 3. [Clash Meta for Android (CMFA)](https://github.com/MetaCubeX/ClashMetaForAndroid/releases)
* **Target Audience:** Android power users and hardcore enthusiasts.
* **Core Advantages:** Based on the powerful Meta (Mihomo) open-source core; has the best compatibility for new protocols and complex network environments; the "perfect score" solution for Android.
* **Tutorial:** [CMFA Quick Start Guide](https://clash.guide/en/clash-meta-for-android.html)

### 4. [Hiddify Next](https://github.com/hiddify/hiddify-app/releases)
* **Target Audience:** Beginners looking for zero-config and users wanting to experience multiple protocols (Sing-box).
* **Core Advantages:** Built on Sing-box; supports an extremely wide range of protocols. The UI is very user-friendly and simplified, truly achieving "one-click connectivity" for average users.

### 5. [ClashMi](https://github.com/KaringX/clashmi/releases)
* **Target Audience:** Users who frequently switch collaboration between mobile and desktop devices.
* **Core Advantages:** Full platform support (including iOS sideloading/TestFlight and Android), lightweight and agile.

## ⚠️ Discontinued Clients (Not Recommended for New Users)

The following classic projects once had massive user bases but have announced they are discontinued due to core developer exits or other reasons. Since they cannot support the latest Mihomo features and new encryption protocols, if you are setting up your environment for the first time, **it is recommended to download projects marked with 🟢 from the table above** to avoid parsing errors and unknown crashes:

- ❌ `Clash for Windows` (CFW) -> Please migrate to **Clash Verge Rev** or **GUI for Clash**
- ❌ `Clash Verge` (Original) -> Please migrate to **Clash Verge Rev**
- ❌ `ClashX` -> Please migrate to **ClashX Meta** or **FlClashX**
- ❌ `Clash for Android` (CFA) -> Please migrate to **Clash Meta for Android**
- ❌ `Nekoray` -> Please migrate to new clients based on Sing-box or Mihomo
